# TFG - Gestión de Inversiones

Proyecto Final de Grado desarrollado con Laravel.

## Tecnologías
- PHP
- Laravel
- MySQL
- Tailwind
- Livewire
- JavaScript
- CSS
- HTML

## Instalación
1. Clonar repositorio
2. composer install
3. cp .env.example .env
4. añadir claves api de coingecko y marketstack (gratuitos)
5. php artisan key:generate
6. php artisan migrate


## Ejecución
1.Ejectar xampp apache y mysql
2.Ejecutar en consola php artisan serve y npm run dev
3.¡Listo!
